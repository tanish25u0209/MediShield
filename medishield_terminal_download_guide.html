1

2
Click "Create New Token" → downloads kaggle.json
3
Move the file and set permissions:
# Linux / macOS
mkdir -p ~/.kaggle
mv ~/Downloads/kaggle.json ~/.kaggle/kaggle.json
chmod 600 ~/.kaggle/kaggle.json

# Windows (PowerShell)
mkdir $env:USERPROFILE\.kaggle
copy $env:USERPROFILE\Downloads\kaggle.json $env:USERPROFILE\.kaggle\kaggle.json

Test it works: Run kaggle datasets list — if it prints dataset names, your key is working. If you see "401 Unauthorized", your kaggle.json is in the wrong place or corrupted.

Download all 6 in one shot (bash script)

#!/bin/bash
declare -a DATASETS=(
  "aryashah2k/mobile-captured-pharmaceutical-medication-packages"
  "vencerlanz09/pharmaceutical-drugs-and-vitamins-synthetic-images"
  "trumedicines/1k-pharmaceutical-pill-image-dataset"
  "trumedicines/pharmaceutical-tablets-dataset"
  "hossam82/pharmacy-products-dataset"
  "aimlveera/counterfeit-product-detection-dataset"
)

for ds in "${DATASETS[@]}"; do
  name=$(echo $ds | cut -d'/' -f2)
  echo "Downloading: $ds"
  kaggle datasets download "$ds" -p "./$name" --unzip
  echo "Done: $name"
done
echo "All datasets downloaded."

Save the above as download_all.sh then run: chmod +x download_all.sh && ./download_all.sh

MVTec AD has a public FTP server — no registration needed, direct wget command works. The full dataset is 4.9 GB. For MediShield you only need the capsule and bottle subsets (~200 MB total).

Option A — Download full MVTec AD (4.9 GB)

wget ftp://guest:GU.205dldo@ftp.softronics.ch/mvtec_anomaly_detection/mvtec_anomaly_detection.tar.xz

# Extract everything
tar -xJf mvtec_anomaly_detection.tar.xz
# Extracts: bottle/ capsule/ hazelnut/ leather/ ... (15 folders)

Option B — Extract only capsule + bottle subsets (recommended)

# Download first (same as above)
wget ftp://guest:GU.205dldo@ftp.softronics.ch/mvtec_anomaly_detection/mvtec_anomaly_detection.tar.xz

# Extract ONLY capsule and bottle folders
tar -xJf mvtec_anomaly_detection.tar.xz \
  mvtec_anomaly_detection/capsule \
  mvtec_anomaly_detection/bottle

# Move to anomaly folder
mv mvtec_anomaly_detection/capsule ../anomaly/mvtec_capsule
mv mvtec_anomaly_detection/bottle ../anomaly/mvtec_bottle

# Folder structure after extraction:
# mvtec_capsule/train/good/      ← clean reference images
# mvtec_capsule/test/crack/      ← cracked capsules (anomaly)
# mvtec_capsule/test/faulty_imprint/  ← label anomaly
# mvtec_capsule/test/squeeze/    ← deformed capsules

License note: MVTec AD is CC BY-NC-SA 4.0 — non-commercial use only. For a hackathon project this is perfectly fine. Do not use it in a commercial product.

NIH C3PI is hosted on data.gov. The sample ZIP (493 MB) is the right choice for hackathon — the full training set is 6.8 GB. Direct wget works, no auth needed.

Option A — NIH C3PI Sample data (493 MB) — recommended

# Direct download from data.gov (no login needed)
wget -O c3pi_sample.zip \
  "https://data.lhncbc.nlm.nih.gov/public/Pills/C3PI/c3piSampleData.zip"

# Unzip
unzip c3pi_sample.zip -d ./nih_c3pi_sample

# Contains ~500 reference pill images (tablets + capsules)

Option B — Reference images only (4,000 high-quality images, 6.8 GB)

# Only run if you want the full high-quality reference set
wget -O c3pi_reference.zip \
  "https://data.lhncbc.nlm.nih.gov/public/Pills/C3PI/C3PI_Test.zip"

unzip c3pi_reference.zip -d ./nih_c3pi_reference

Remove duplicate images (hash-based)

pip install imagededup

python3 -c "
from imagededup.methods import PHash
phasher = PHash()

for folder in ['dataset1','dataset2','dataset3','dataset4','dataset5']:
    dupes = phasher.find_duplicates(image_dir=f'./{folder}', max_distance_threshold=10)
    total = sum(len(v) for v in dupes.values())
    print(f'{folder}: {total} duplicates found')
"

Check minimum image size (filter out tiny images)

python3 -c "
from PIL import Image
import os, glob

too_small = []
for path in glob.glob('./dataset*/**/*.jpg', recursive=True) + \
            glob.glob('./dataset*/**/*.png', recursive=True):
    try:
        w, h = Image.open(path).size
        if w < 80 or h < 80:
            too_small.append(path)
    except: pass

print(f'Images too small (<80px): {len(too_small)}')
for p in too_small[:10]:
    print(' ', p)
"

Final dataset summary count

python3 -c "
import os, glob

classes = ['tablet','capsule','syrup','injection','other']
base = '../raw'

print('Class         | Count | Status')
print('-'*40)
for cls in classes:
    path = os.path.join(base, cls)
    if os.path.exists(path):
        n = len(glob.glob(f'{path}/*.jpg') + glob.glob(f'{path}/*.png'))
        status = 'OK' if n >= 50 else 'NEEDS MORE'
        print(f'{cls:<14}| {n:<6}| {status}')
    else:
        print(f'{cls:<14}| 0     | MISSING FOLDER')
"
